# 水泳練習記録
## サイト概要
### サイトテーマ
水泳の練習記録を共有し、成長を可視化できるコミュニティサイト
​
### テーマを選んだ理由
私は5歳から小学6年生まで水泳を続けてきました。  
当時は練習記録を紙で管理しており、過去の記録を振り返ることや成長の推移を把握することが難しいと感じていました。  
また、大人になってから運動記録を振り返る機会があり、継続管理の重要性を改めて実感しました。  
そこで、練習データを一元管理し、数値やグラフで可視化できる仕組みを提供したいと考えました。  
さらに、他の利用者と記録を共有できる環境を整えることで、成長実感とモチベーション向上を支援したいと考え、このテーマにしました。

### ターゲットユーザ
・水泳を習慣的に行っている個人練習者  
・フィットネスジムなどで継続的にトレーニングを行っている人
・タイムや距離を記録し、目標達成を目指している人  
・紙やメモアプリでの記録管理に不便さを感じている人  
　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　
### 主な利用シーン
・水泳やトレーニング後に、その日の練習内容（距離・タイム）を記録する時  
・過去の記録を振り返り、成長の推移を確認したい時  
・目標タイムや月間距離を管理し、達成状況を確認する時  
・他のユーザーの公開記録を閲覧し、モチベーション向上につなげたい時
